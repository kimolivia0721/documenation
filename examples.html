# js-library-kimheeli
PeopleGenerator has one attribute, people which contains lists. In each item of this list, it contains html format of person, boolean, boolean, name of the person, height for reference when chaning poses, emoji source respectively.

## makePeople()
This function will generate a person and add to people list.

## changeHairColor(index, color)
Change the color of hair of the specified people[index] and to the "color"

## changeSkinColor(index, color)
Similar with changeHairColor

## changePose(index)
Change the pose of people[index]

## changeGender(index)
default set to male, call once to change to female, call once more to change to not specified, call again to go back to male.

## changeName(index, name)
Change the name of people[index]


## changeHeight(index, height)
change the height of people[index] by multiplicator, height.


## changeEmoji(index, emoji)
change the mood for people[index]
0: smiley 1: :P 2: angry 3: hmm 4: heart eyes
